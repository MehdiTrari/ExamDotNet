# ExamDotNet

Ce depot contient deux projets distincts pour les exercices.

## 1) API e-commerce (Exercices 1 et 3)
- Dossier : `ECommerceApi`
- Contenu : API REST + BDD EF Core InMemory
- Demarrage :
  ```bash
  dotnet run --project /home/mehdi/ExamDotNet/ECommerceApi/src/ECommerceApi
  ```
- Tests :
  ```bash
  dotnet test /home/mehdi/ExamDotNet/ECommerceApi/ECommerceApi.sln
  ```

## 2) Labyrinthe (Exercice 2)
- Dossier : `Labyrinthe`
- Contenu : algo + runner console + tests TDD
- Demarrage :
  ```bash
  dotnet run --project /home/mehdi/ExamDotNet/Labyrinthe/src/MazeRunner
  ```
- Tests :
  ```bash
  dotnet test /home/mehdi/ExamDotNet/Labyrinthe/Labyrinthe.sln
  ```
